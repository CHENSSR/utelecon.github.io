以下の手順は3月中旬に予定されているシステム変更の後のものです．現在の手順は「Webexアカウントの作成とログイン」のページをご覧ください．

  • 確認：サインインしてみる
    1. 東京大学のWebexページにアクセスして，右上の「サインイン」を押してください．
    2. 1234567890@utac.u-tokyo.ac.jp のように，10桁の共通ID（UTokyo Accountのユーザ名）の後に東京大学であることを表す記号 @utac.u-tokyo.ac.jp を付けたものを入力してください．
    3. UTASなどと同じような東京大学のユーザ名とパスワードを入力する画面が出たら，UTokyo Account{% if page.slug != "oc" and page.slug != "faculty_members" %}（10桁の共通ID）{% endif %}の情報を入力してください（この画面は，既に入力済みなら出ないこともあります）．
    4. 自分の名前や「ミーティングを開始する」ボタンが表示されている画面になればOKです．
    • この手順がうまくいかないときはサポート窓口に相談してください．
  • 作業：パソコンにアプリをインストールする
    1. Webexのダウンロードページにアクセスしてください．
    2. 一番上の「Cisco Webex Meetings デスクトップアプリ」にある「ダウンロード」ボタンを押してください．
    3. 以後の手順は場合によって異なります．インストールの完了まで自動で進むこともあれば，確認ボタンを押したり，ダウンロードしたファイルを自分で開いたりといった操作の必要があるかもしれません．
    • この手順がうまくいかないときはサポート窓口に相談してください．
  • 任意の作業：スマホにアプリをインストールする
    1. Androidの人はGoogle Playから，iPhoneの人はApp Storeからインストールしてください．
  • 確認：東大のアカウント専用の会議室に入ってみる
    • 確認用の会議室を現在準備中です．
  • 任意の作業：アプリにサインインする{% if page.slug == "oc" %}（自分で会議を開く機会がある人などには便利です）{% endif %}
    1. Webexアプリを起動し，「サインイン」ボタンを押してください．すでに他のアカウントでサインインしている場合は，事前にサインアウトをしてください．
    2. 1234567890@utac.u-tokyo.ac.jp のように，10桁の共通ID（UTokyo Accountのユーザ名）の後に東京大学であることを表す記号 @utac.u-tokyo.ac.jp を付けたものを入力してください．
    3. UTASなどと同じような東京大学のユーザ名とパスワードを入力する画面が出たら，UTokyo Account{% if page.slug != "oc" and page.slug != "faculty_members" %}（10桁の共通ID）{% endif %}の情報を入力してください（この画面は，既に入力済みなら出ないこともあります）．
    4. 自分の名前や「ミーティング開始」ボタンが表示されている画面になればOKです．
    • {% if page.slug != "oc" %}
  • 練習：会議室を作ってみる
    1. 東京大学のWebexページにアクセスしてください．サインインしていない場合は，サインインしてください．
    2. 「スケジュールする」ボタンを押してください（「ミーティングを開始する」ボタンの右隣にあります）．
    3. 会議室の詳細を設定する画面が表示されます．練習なので，よくわからない項目は飛ばすなど適当に入力して，一番下の「保存」を押してください．
    4. 作成された会議室の情報を確認できる画面に移動します．「招待リンク」欄に会議室のURLが表示されており，このURLから会議室に入室できるようになっています．
    • この手順がうまくいかないときはサポート窓口に相談してください．
