    1. Log in to ITC-LMS

    • Troubleshooting guide
      • The video is not displayed or does not play: This problem occurs when using the Safari browser. Please use other browsers, such as Google Chrome, Firefox, or Microsoft Edge.
      • Neither "Passed" nor "Failed" is displayed after the test: Access the top page of ITC-LMS. If "ECCS Seminar" is no longer displayed, you have passed the test.
      If the issue is still not resolved, please consult the Technical Support Desk.

  • Exercise: Log in
    • For this procedure, please use ECCS terminals installed on designated locations on campus (in Japanese).
    1. Please wait for an hour for the system to update after you have passed the "New User Information Seminar".
    2. Press the power button on the ECCS terminal to turn it on.
    3. In the log-in screen that appears, enter your UTokyo Account username (10-digit number) and password.
    • If this procedure does not work, please consult the ECCS tutors (student staff) who are available at the main terminal locations.
