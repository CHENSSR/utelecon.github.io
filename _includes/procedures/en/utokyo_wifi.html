  • Task: Try Connecting to the WiFi
    • This task can only be done on campus where UTokyo WiFi is available.
    1. The name (SSID) of the UTokyo WiFi service is UTokyo-WiFi. Please select this name on the Wi-Fi connection screen of your device (smartphone or computer).
    2. Depending on the device, additional steps may be required, such as checking the certificate.
    3. When asked to enter your user ID and password, enter the user ID and password in the notification email you received after application (not your UTokyo Account information).
    4. Once the connection is established, you can use the Wi-Fi services normally.
    • Troubleshooting guide
      • If the network is weak, you may get disconnected during the task. Please try again at a different location or time of day.
      • The user ID is an 8-digit alphanumeric character starting with the letter "s" or "w". Ensure to include letters in the alphabet.
      • Ensure that you entered the password correctly. Try copying and pasting the password from the notification email.
      • If you are copying and pasting from the notification email, ensure that there are no extra spaces before or after the copied characters.
      • If you have applied for an account more than once, only one of the accounts will be valid. Please use the accounts listed on the UTokyo WiFi account menu only.
      If the issue is still not resolved, please consult one of following support desks for assistance. There are ECCS tutors (student staff) available at various locations on campus. You can ask them for help in person. The Technical Support Desk can also be consulted on this website.
