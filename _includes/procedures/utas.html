  • 練習：ログインしてみる
    1. UTASにアクセスして，「ログイン」を押してください．
    2. {% if page.procedure == "oc" or page.procedure == "faculty_members" %}ユーザ名とパスワードの入力を求められるので，UTokyo Accountのユーザ名（10桁の数字）とパスワード（自分で設定したもの）を入力してください．{% else %}ユーザ名とパスワードの入力を求められたら，UTokyo Account（10桁の共通ID）の情報を入力してください．{% endif %}
    • この手順がうまくいかないときはサポート窓口に相談してください．
    • {% if page.procedure != "faculty_members" %}
  • 確認：メールアドレスが登録されているか確認する{% if page.procedure != "oc" %}（学生のみ）{% endif %}
    1. ログインした後，上部のメニューで「学生情報」「現住所等変更入力」を順に押してください．
    2. その画面の「E-MAIL 1」という欄に，あなたに確実に届くメールアドレスが登録されているか確認してください．
    3. もし，何も登録されていなかったり，メールアドレスが間違っていたりしたら，あなたに確実に届くメールアドレスに変更してください．
    • この手順がうまくいかないときはサポート窓口に相談してください．
    • {% endif %}
