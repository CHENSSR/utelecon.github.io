  3. 既にUTokyo Accountにサインイン済みの場合を除き，UTokyo Accountのサインイン画面が表示されるので，サインインしてください．
  4. 「新規申請」ボタンを押してください．
  5. 「通知用メールアドレス」の欄に，{% if page.slug == "oc" %}UTASで前日までに登録したメールアドレス{% else %}手順2で前日までに登録したメールアドレス{% endif %}が表示されているので，確認してください．このメールアドレスに，UTokyo WiFiの利用に必要なアカウント情報（ユーザーID・パスワード）が書かれた通知メールが届きます．
  6. 「申請」ボタンを押してください．
  7. しばらく待つと通知メールが届きます．
  • この手順がうまくいかないときのヘルプ
    • 以下のような画面が出てアカウントメニューにアクセスできない場合 情報セキュリティ教育を受講していないため，アカウントが停止されています．詳細は{% if page.slug == "oc" %}UTokyo WiFiのページの「情報セキュリティ教育」{% else %}「情報セキュリティ教育」{% endif %}を参照してください．
    • 通知メールが届かない場合：
      • 迷惑メールフォルダなどに紛れ込んでいないかどうか，再度確認してください．
      • メールは，申請画面の「通知用メールアドレス」欄に表示されているメールアドレス（{% if page.slug == "oc" %}前日までにUTASに{% else %}手順2で前日までに{% endif %}登録したもの）に届きます．違うメールアドレスを見ていないか確認してください．
    これらを確認して，それでもうまくいかなければ，サポート窓口に相談してください．
